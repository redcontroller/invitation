# 파티 참가신청 페이지
### 멋사 프론트엔드 스쿨 BBQ 파티 초대장
- 개발 목적 : **손쉬운 파티 초대와 응답**을 받을 수 있다.
- 기술 스택 : HTML/CSS, Vanilla JavaScript
- 사용된 API : Kakao Maps, emailJS
- 웹 호스팅 서버 : Netlify
- 실행 환경 :
  - Windows 10
  - Chrome 116.0.5845.111 (공식 빌드) (64비트)
- 주요 구현 항목 :
  - [x] 파티정보 전달
  - [x] 입력정보 누락 알림
  - [x] 참가신청 Email 수신
  - [x] 자동답변 Email 발신
  - [x] 카카오 지도 API 연동
  - [x] 모바일 반응형 웹 페이지
